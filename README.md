## Expressions 是个什么东西？

Expressions 是 Apptorium 出品的一款 macOS 专属正则表达式测试工具。界面极简，操作直接——上面输规则，下面贴测试文本，匹配结果实时刷新，根本不用点任何按钮。

对，就这么简单。

当然，"简单"背后是有讲究的。Expressions 并不是一个玩具级工具，它支持 ICU 正则标准——这正是苹果 Foundation 框架用的那套，iOS/macOS 开发者写完就能直接用，不用再考虑"这个环境支不支持"。

## 用起来长什么样

打开软件，界面干净到有点不像话。没有工具栏，没有一堆设置按钮，两个文本区域就是全部。但这恰恰是让人用得进去的原因——你的注意力全在正则本身，不会被界面分散。

**语法高亮**是第一个让人安静下来的功能。规则写出来是带颜色的，不同分组用不同颜色区分，一眼就能看清楚结构在哪里、哪块出了问题。高亮不只在测试文本那边，规则本身也是高亮的，这点和不少在线工具不一样。

**实时预览**这个没什么好说的，改一个字符结果马上变，不需要手动触发任何操作。调试的时候那种即时反馈的感觉，比来回切窗口舒服太多。

**表达式收藏夹**是一个用多了才会发现真香的功能。写好一个常用的规则，`⌘ L` 收藏，下次直接调出来改改用。不用再翻以前的代码找"我上次写的那个邮箱匹配是什么来着"。

**内置语法参照表**解决了正则最常见的痛点：语法记不住。`⌘ R` 快捷键弹出来，各种元字符、量词、分组一目了然，查完继续写，不用开浏览器。

此外还有多行模式、允许空白字符与注释、全屏专注模式，以及跟随系统的深色/浅色主题自动切换。

## 哪些人适合用

说实话，这款工具的受众挺宽的：

- **开发者**：写 iOS/macOS 应用的话，ICU 标准直接对口，测完就能用
- **数据处理**：经常要用脚本处理文本的同学，有个顺手的测试环境很重要
- **正则新手**：内置语法表降低了学习门槛，边看边试，上手比去网页查快
- **不想用网页版的人**：正则测试网站虽然多，但要开浏览器、要联网，Expressions 在本地就跑，快，也省事

唯一的限制是平台——目前只有 macOS 版本，需要 macOS 10.11 或更新系统，同时完整支持 Apple Silicon。最新版本 1.3.9 已于 2025 年 12 月更新，适配了 macOS Tahoe。
